🎟️ Eventora

A full-stack event booking platform built using the MERN stack. It focuses on secure authentication, smooth booking workflows, and handling real-world challenges like overbooking and role-based access.

🚀 Features
Secure authentication using JWT
OTP-based email verification (2FA)
Email handling using Nodemailer (login, registration, notifications)
Event browsing and booking system
Admin can create new events
Admin can accept or reject booking requests
Overbooking prevention using queue logic
Email notifications for booking updates and confirmations
Analytics dashboard for tracking bookings and activity
Role-based access (Admin / User)

🛠️ Tech Stack

Frontend:
React, Tailwind CSS, shadcn/ui

Backend:
Node.js, Express.js

Database:
MongoDB (Mongoose)

Other:
JWT, Nodemailer

💡 Highlights
Handles concurrent booking scenarios efficiently
Implements real-world authentication flow
Includes admin-level control over bookings and events
Clean and responsive UI with modern design

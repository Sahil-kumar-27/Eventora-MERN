## ✨ Overview

Eventora allows users to browse and book events while providing admins with full control over event management and booking approvals.  
The system is built with scalability and real-world constraints in mind, including concurrent booking handling and secure user flows.

---

## 🚀 Features

- 🔐 JWT-based authentication  
- 📩 OTP-based email verification (2FA)  
- 📧 Email services using Nodemailer (login, registration, notifications)  
- 🎟️ Event browsing and booking system  
- ⚙️ Admin dashboard for managing events  
- ➕ Admin can create new events  
- ✅ Admin can approve or reject booking requests  
- 🚫 Overbooking prevention using queue logic  
- 📊 Analytics dashboard (bookings, revenue, activity)  
- 👥 Role-based access control (Admin / User)
